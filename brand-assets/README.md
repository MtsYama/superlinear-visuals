# Brand Assets（共用视觉件）

logo、banner、社区头图、会员视觉、社交/合作物料等可复用视觉件。

## 规划结构

```
brand-assets/
├── logo/            # symbol / wordmark / favicon（黑白双版 SVG+PNG）
├── banners/         # 社区 / 课程 / 活动 banner
├── space-headers/   # 7 分区 space 头图（可复用 space-illustrations 输出）
├── membership/      # Stay Superlinear 会员视觉件
└── social/          # 对外合作 / 播客等衍生模板
```

- 配色与字体规范见 `../brand-core/`。
- 所有进入本目录的 PNG 自动获得公开直链（规则见仓库根 README）。
